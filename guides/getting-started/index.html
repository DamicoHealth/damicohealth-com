Getting Started

For first-time users. Get the app running, complete setup, and document your first patient encounter.

1. Download the app

DH Field EMR runs on iPhone, iPad, and Mac. Download it from the Apple App Store.

Download on App Store

Tip

The app is free to download and includes a free tier with no time limit. You can upgrade your tier later from inside the app if your team grows.

2. First launch and Terms agreement

The first time you open DH Field EMR, you'll see a Terms of Service screen. Read through and tap Agree to continue. This covers:

  • The app is for international medical outreach only — not for use within US healthcare
  • It is not HIPAA compliant and not a certified EHR
  • Your organization is responsible for patient consent and local data protection laws
  • All clinical decisions are made by qualified clinicians, not the app
Important

If your work involves US-based patients or any setting that requires HIPAA compliance, this app is not appropriate for you. See the Terms of Service for full details.

3. Choose a setup path

After agreeing to the Terms, the setup wizard offers three options. Pick the one that matches your situation.

Standalone

Best for: a solo clinician on a single device, or trying out the app.

  • All data lives on this device only
  • No cloud sync
  • No organization code needed
  • You can switch to an organization later from the Admin screen

Create Organization

Best for: the first device for a team. Whoever sets this up becomes the admin.

  • Choose an organization name
  • The app generates a unique Organization Code (e.g. HOPE-2026-XQ7K)
  • Set an admin password to protect customization and configuration
  • Connect to the cloud so other devices can join
Save your Organization Code

Write it down somewhere safe. You'll need to give it to teammates who join your organization later.

Join Organization

Best for: additional devices joining a team that already has an admin device set up.

  • Get the Organization Code from your admin
  • Enter it on the Join screen
  • The app downloads your team's presets and history
  • You start as a Standard device — your admin can promote you if needed

4. Set your device name

Give this device a name so the rest of your team can tell it apart in sync logs and audit trails. Examples:

  • Triage iPad 1
  • Dr. Okafor's iPhone
  • Backend Mac

You can change this later in Admin → Device Name.

5. The main interface

Once setup is complete, you'll see the main app. There are five sections in the top navigation:

Records

Where most of your work happens. Create new encounters, search past records, view patient history, and mark referrals complete.

Analytics

Real-time dashboards showing patient volume, disease burden, lab positivity rates, provider productivity, and donor-ready summary reports.

Scheduling

Surgery day planning and referral tracking. See who's scheduled and where, and mark surgeries complete.

Options

Customize presets — sites, physicians, medications, diagnoses, lab tests, prescription bundles, referral types, and more. Available to admin devices.

Admin

Cloud Connection, sync controls, database backup, donor report generation, subscription tier management, and device settings. Admin devices have the full version; standard devices see a reduced subset.

6. Create your first patient encounter

  1. Tap Records
  2. Tap + New Encounter
  3. Enter patient first name, last name, and date of birth — the MRN auto-generates
  4. Choose a site and a provider
  5. Enter chief complaint (or pick from presets)
  6. Fill out vitals (temperature, blood pressure, weight)
  7. Record any labs as POS / NEG or numeric values
  8. Pick a diagnosis from presets, or type a custom one
  9. Add medications, procedures, imaging, or referrals as needed
  10. Tap Save
Tip — Save & Next

For high-volume clinic days, use Save & Next instead of Save. It saves the current encounter and immediately opens a fresh form so you can roll through patients without going back to the records list every time.
